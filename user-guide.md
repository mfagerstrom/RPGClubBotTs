# RPGClub Bot User Guide

Welcome to the RPGClub bot. This guide explains what the bot can do and how to use it in simple terms.

## Quick Tips

- Use slash commands by typing `/` in Discord and picking a command.
- Many replies are private to you unless you choose to show them in chat.
- If you are not sure, start with `/help` and use the menu.

## Now Playing

Track the games you are currently playing.

- **View your list**
  - Use `/now-playing list`.
  - If you have no games yet, the bot will show a welcome message and a button to add your first game.
- **Add a game**
  - Open your list, then use the **Add Game** button.
  - The bot will guide you through a search and add it to your list.
- **Edit your note**
  - Use the **Edit Note** button from your list.
- **Sort your list**
  - Use the **Sort** button to move games up one by one.
- **Remove a game**
  - Use the **Remove Game** button from your list.
- **Add a completion**
  - Use the **Add Completion** button from your list.
  - You can choose the completion type and whether to announce it.

## Game Completions

Keep track of games you finished.

- **Add a completion**
  - Use `/game-completion add`.
  - You can include date, note, and playtime.
- **List completions**
  - Use `/game-completion list`.
  - You can filter by year or title.
- **Edit or delete**
  - Use `/game-completion edit` or `/game-completion delete`.

## GameDB

Search for games and view details.

- **Search**
  - Use `/gamedb search title:<game name>`.
- **View a game**
  - Use `/gamedb view title:<game name>`.
- **Import**
  - If a game is missing, the bot can import from IGDB when prompted.

## Profile

View and edit your profile info and linked accounts.

- **View your profile**
  - Use `/profile view`.
- **Edit your profile**
  - Use `/profile edit` to update your links and info.
- **Search profiles**
  - Use `/profile search` to find members.

## Reminders

Set reminders so the bot can message you later.

- **Create a reminder**
  - Use `/remindme create`.
- **List or delete**
  - Use `/remindme menu` to manage reminders.

## GOTM and NR-GOTM

Vote and nominate games for Game of the Month and Non RPG of the Month.

- **See current info**
  - Use `/round` or `/nextvote`.
- **See nominations**
  - Use `/noms`.
- **Nominate a game**
  - Use `/gotm nominate` or `/nr-gotm nominate`.

## HLTB

Check estimated playtime for a game.

- Use `/hltb title:<game name>`.

## Suggestions

Send a suggestion to the bot owner.

- Use `/suggestion title:<idea>`.

## Game Giveaways

Claim or donate game keys.

- Use `/gamegiveaway` to open the giveaway hub.
- Claims are sent to you by direct message.

## Logging Features

The bot posts logs in staff channels for:

- Message edits and deletes
- Role changes
- Nickname and name changes
- Avatar changes
- Joins, leaves, kicks, bans, and unbans
- Server and channel updates

These are automatic and you do not need to do anything.

## If Something Looks Wrong

- Try the command again.
- Use `/help` to look up the command.
- Ask a moderator if you need help.
